# Rooti
 
A biologically-inspired tree simulation that grows on at transparent, click-through overlay on your Windows desktop.
 
---
 
## What is this?
 
`Rooti` is a living desktop ornament. A procedurally generated tree that grows organically in the background of your screen, abstracting real biological processes like the regulation of growth with auxin, while remaining fully transparent and non-intrusive. You can click through it, work normally, and just glance over occasionally to see how your tree is doing.

Inspired by the desktop pets of old, this project serves no functional purpose and is fully aesthetic, ornamental, and just for fun!

Ultimately, this is a passion project to help the developer learn Rust!
